### 1. 大括号风格要求
one true brace style 将大括号放在控制语句或声明语句同一行的位置, 不允许块的开括号和闭括号在同一行

- ✗ 不推荐

```js
function foo()
{
  return true
}

function nop() { return }

```

- ✓ 推荐

```js
function foo() {
  return true
}

function nop() { 
    return
}
```

### 2. 使用驼峰命名法（Object key 除外）

- ✗ 不推荐

```js
// 包导入
import { no_camelcased } from "external-module"

// 变量定义
var my_favorite_color = "#112C85";

// 函数定义
function do_something() {
    // ...
}

```

- ✓ 推荐

```js
// 包导入
import { no_camelcased as camelCased } from "external-module"

// 变量定义
var myFavoriteColor = "#112C85";

// 函数定义
function doSomething() {
    // ...
}

// Object key 允许非驼峰命名
var obj = {
    my_pref: 1
}
```

### 3. 【建议】缩进使用 4 个空格，并且 switch 语句中的 Case 需要缩进

- ✗ 不推荐

```js
if (a) {
  var b = a
  function foo(c) {
    var d = c
  }
}

```

- ✓ 推荐

```js
if (a) {
    var b = a
    function foo(c) {
        var d = c
    }
}

```

### 8. 禁止空语句（可在空语句写注释避免），允许空的 catch 语句

- ✗ 不推荐

```js
if (foo) {
}

while (foo) {
}

switch(foo) {
}

try {
    doSomething();
} catch(ex) {

} finally {

}
```

- ✓ 推荐

```js
if (foo) {
    // empty
}

while (foo) {
    /* empty */
}

try {
    doSomething();
} catch (ex) {
    // continue regardless of error
}

try {
    doSomething();
} finally {
    /* continue regardless of error */
}
```

```js
    // ...
```
```js
})()
```

### 10. 禁用不必要的分号

- ✗ 不推荐

```js

    // code
};
```

- ✓ 推荐

```js

    // code
```

### 11. generator 的 * 前面禁止有空格，后面必须有空格

- ✗ 不推荐

```js

var anonymous = function *() {};

var shorthand = { *generator() {} };
```

- ✓ 推荐

```js
function* generator() {}

var anonymous = function* () {};

var shorthand = { * generator() {} };
```

- ✓ 推荐

```js
 */
```

### 16. 如果一个变量不会被重新赋值，必须使用 `const` 进行声明

- ✗ 不推荐
```js
let a = 3;
console.log(a);

let a;
a = 0;
console.log(a);

// `i` is redefined (not reassigned) on each loop step.
for (let i in [1, 2, 3]) {
    console.log(i);
}

// `a` is redefined (not reassigned) on each loop step.
for (let a of [1, 2, 3]) {
    console.log(a);
}
``` 

- ✓ 推荐
  
```js
const a = 0;

// it's never initialized.
let a;
console.log(a);

// it's reassigned after initialized.
let a;
a = 0;
a = 1;
console.log(a);
```

### 20. 要求使用 === 和 !== 代替常规的相等运算符 == 和 !=

```js
  // ✓ 推荐 
  a === b
  foo === true
  bananas !== 1
  value === undefined

  // ✗ 不推荐
  a == b
  foo == true
  bananas != 1
  value == undefined
```

### 23. 对象写在一行时，大括号里旁边不需要空格

```html

 /* ✗ 不推荐 */
```

### 25. new, delete, typeof, void, yield 等后面必须有空格，一元操作符 -, +, --, ++, !, !! 禁止有空格

```html

new Foo;

void 0;

++foo;

foo--;

-foo;

+"3";

 /* ✗ 不推荐 */

void{foo:0};

new[foo][0];

delete(foo.bar);

++ foo;

foo --;

- foo;

+ "3";
</script>
```
